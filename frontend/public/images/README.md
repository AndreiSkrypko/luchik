# Папка для изображений

## Загрузка логотипа

1. Поместите ваш файл логотипа в эту папку
2. Назовите файл `logo.png` (или `logo.jpg`, `logo.svg`)
3. Рекомендуемый размер: 200x200 пикселей или больше
4. Формат: PNG, JPG или SVG

## Поддерживаемые форматы:
- `.png` - рекомендуется для логотипов с прозрачностью
- `.jpg` - для фотографий
- `.svg` - векторные изображения (масштабируются без потери качества)

После загрузки файла логотип автоматически отобразится на сайте!
